### 4.2 Station 数据源详情（新发现）

**路径**：`data/substation_coordinates/`

**文件结构**：
```json
{
  "page_name": "substationCoordinates",
  "raw_data": [
    {
      "id": "1826894205214539777",
      "prjCode": "1516A023001U",
      "singleProjectCode": "1516A023001U01",
      "longitude": "112.81171292",
      "latitude": "28.24226858"
    }
  ]
}
```

**字段映射**：
| 源字段 | 目标字段 | 说明 |
|--------|----------|------|
| `singleProjectCode` | `single_project_code` | 主键 |
| `prjCode` | `prj_code` | 项目编码 |
| `longitude` | `longitude` | 经度 |
| `latitude` | `latitude` | 纬度 |
| 无 | `name` | 使用 `single_project_code` 作为显示名 |

### 4.3 Unresolved 映射策略

**方案**：映射占位层（只读）

**实现**：
- 新增 `YearProgressMapping` 模型
- 字段：`source_code`, `single_project_code`, `status`（unresolved/resolved）
- 接口：`/api/map/summary` 增加 `unresolved_count`
- UI：详情抽屉显示 unresolved 标记

**不做的**：
- 人工编辑映射
- 批量解析工具
- 复杂映射算法

### 4.4 详情抽屉设计

**触发方式**：点击地图任意对象（Line/Tower/WorkPoint/Station）

**展示内容**：
| 字段 | Line | Tower | WorkPoint | Station |
|------|------|-------|-----------|---------|
| 对象类型 | ✅ | ✅ | ✅ | ✅ |
| 主键/编码 | `single_project_code` | `tower_no` | `meeting_id` | `single_project_code` |
| 名称 | `single_project_code` | `tower_no` | `prj_name` | `single_project_code` |
| 坐标 | - | ✅ | ✅ | ✅ |
| 所属项目 | - | `single_project_code` | `single_project_code` | `prj_code` |
| unresolved 标记 | - | - | - | - |

**不做**：
- 复杂 tabs
- 图表
- 编辑功能

### 4.5 图层显隐控制

**位置**：左侧筛选面板底部

**开关**：
- [x] 线路（Line）
