## 最终修正版：项目生命周期与多武将体系

### 一、项目生命周期模型（核心修正）

```
项目生命周期（一次性）：
┌─────────────┐    ┌─────────────┐    ┌─────────────┐    ┌─────────────┐
│   筹备期    │ → │   建设中    │ → │   验收期    │ → │   已投产    │
│  (未开工)   │    │  (在建)     │    │  (收尾)     │    │  (完结)     │
└─────────────┘    └─────────────┘    └─────────────┘    └─────────────┘
       │                  │                  │                  │
       ▼                  ▼                  ▼                  ▼
   军师制定计划      武将带兵施工       准备投产交接       赛季结算时
   发布开工目标      进度持续推进       达成投产目标       计入战绩

关键结算节点：
├─ 开工结算：项目正式开工时，军师获得"开工达成"战绩
├─ 过程节点：季度/月度过程评价（非结算，仅反馈）
└─ 投产结算：项目投产时，军师和武将分别获得"投产达成"战绩

赛季目标（年初发布）：
├─ 开工目标：计划新开工X个项目
├─ 投产目标：计划投产Y个项目（容量Z MVA）
└─ 过程目标：安全/质量/进度控制指标
```

#### 2.2 武将独立评价体系

```
施工单位武将评价体系（通用）：

维度1：进度执行
├─ 节点达成率：计划节点按时完成比例
├─ 工期控制：实际工期 vs 计划工期
└─ 响应速度：军师军令的响应和执行速度

维度2：质量管控
├─ 一次通过率：质量验评一次通过比例
├─ 返工率：因质量问题返工的比例
└─ 工艺水平：工艺评分（如有）

维度3：安全绩效
├─ 零事故天数：连续安全生产天数
├─ 风险管控：危险点识别和管控效果
└─ 违章率：现场违章次数

维度4：资源效率
├─ 人员效率：人均产出（如人·日/基塔）
├─ 资源利用率：设备、材料使用效率
└─ 成本控制：预算执行情况

评级计算：
总评分 = Σ(各维度得分 × 维度权重)
评级：S(95+) / A(85-94) / B(75-84) / C(60-74) / D(<60)
```

```
建管单位军师评价体系（独立）：

维度1：项目策划
├─ 开工达成率：计划开工项目实际开工比例
├─ 计划准确性：开工时间与计划偏差
└─ 资源预配：人、材、机提前到位率

维度2：协调效率
├─ 问题解决：项目推进中问题的响应和解决速度
├─ 跨单位协调：多军团协作项目的协调效果
└─ 变更控制：设计变更对进度的影响控制

维度3：投资控制
├─ 预算执行：实际投资 vs 预算
├─ 变更成本：变更导致的成本增加比例
└─ 结算效率：竣工结算及时性

维度4：整体达成
├─ 投产达成率：计划投产项目实际投产比例
├─ 质量一次投运：投产一次成功率
└─ 安全零事故：管辖项目整体安全记录
```

├─ 已投产项目：完整结算（开工战绩 + 投产战绩）
