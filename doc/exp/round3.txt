## 修正版：指挥-作战双体系模型

### 一、双体系架构

```
┌─────────────────────────────────────────────────────────────────────┐
│                         指挥体系（Command）                          │
│                      建管单位 = 指挥部/军师府                         │
├─────────────────────────────────────────────────────────────────────┤
│  军师（业主项目经理）                                                │
│  ├── 角色定位：运筹帷幄，决胜千里                                    │
│  ├── 无权直接参与施工（无士兵）                                      │
│  ├── 职责：项目策划、资源协调、进度督导、质量把控                      │
│  └── 战绩维度：项目整体达成率、投资控制、安全零事故、投产一次成功       │
│                                                                     │
│  指挥单元特性：                                                       │
│  ├── 无士兵（不直接参与作战）                                        │
│  ├── 可调度多个作战单位（可任命多个施工单位）                        │
│  ├── 通过"军令"影响战局（发布节点目标、调整优先级）                  │
│  └── 评价基于项目整体结果（而非单点战斗）                            │
└─────────────────────────────────────────────────────────────────────┘
                                    ↓ 任命/指挥
┌─────────────────────────────────────────────────────────────────────┐
│                         作战体系（Combat）                           │
│                      施工单位 = 军团/野战军                           │
├─────────────────────────────────────────────────────────────────────┤
│  武将（施工项目经理）                                                │
│  ├── 角色定位：身先士卒，冲锋陷阵                                    │
│  ├── 直接带兵作战（有士兵）                                          │
│  ├── 职责：现场施工、人员组织、进度执行、质量自检                      │
│  └── 战绩维度：节点达成率、作业效率、质量一次通过率、人员稳定性         │
│                                                                     │
│  士兵（作业人员）                                                    │
│  ├── 直接参与建设（杆塔、线路、变电站）                               │
│  ├── 有准入资格要求                                                  │
│  ├── 可在军团间流动（支援、借调）                                    │
│  └── 个体有技能标签和经验值                                          │
│                                                                     │
│  作战单元特性：                                                       │
│  ├── 直接产出建设进度                                                │
│  ├── 接受军师指挥（但听令于自己军团长）                              │
│  └── 评价基于执行效率和战斗成果                                      │
└─────────────────────────────────────────────────────────────────────┘
```
