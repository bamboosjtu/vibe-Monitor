## 5. 常见问题

### 页面没有数据

先查 DataHub：

```powershell
Invoke-RestMethod -Method Get -Uri "http://localhost:8000/api/v1/sandbox/dates" |
ConvertTo-Json -Depth 10
```

如果 `count = 0`，说明 DataHub 尚无 canonical work_point，需要先从 downloader 同步并运行 processing。

### 浏览器 CORS 错误

DataHub 当前应允许：

```text
http://localhost:5173
http://127.0.0.1:5173
http://localhost:4173
http://127.0.0.1:4173
```

如果 Vite 端口变化，需要同步调整 DataHub CORS 配置或固定 Vite 端口。

### backend 与 DataHub 端口

- DataHub 默认端口：`8000`
- backend 默认端口：`8001`

两者不会冲突。如需同时运行，请确认各自端口配置。
