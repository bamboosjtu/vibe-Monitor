## 三、数据流

### Station 数据完整链路

```
data/substation_coordinates/*.json
    → StationImporter (backend)
        → raw_station (raw_data JSON)
        → station_current (解析后的结构化数据)
            → GET /api/map/skeleton
                → getSkeleton() (frontend API)
                    → loadSkeletonLayers() (MapContainer)
                        → station-point-data (GeoJSON Source)
                        → station-point-layer (Symbol Layer)
```

### 点击交互链路

```
用户点击地图对象
    → MapContainer click handler
        → setSelectedObject({ type, data })
            → DetailPanel 读取 selectedObject
                → 按 type 渲染对应字段
```
